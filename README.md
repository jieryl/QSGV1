# QSGV1.0 - 迁善公宗族管理系统
迁善公总谱编撰委员会 | 数字化宗族档案管理平台

---

## 项目简介
本项目是基于 GitHub 与 Python 构建的**低代码、插件化宗族管理系统**，核心目标是实现迁善公族谱资料的数字化存档、协作校对与版本化管理，为后续数据治理与应用扩展提供安全可靠的磐石底座。

### 核心架构哲学
- 磐石内核：以 DB1.5.7D 为永久冻结底座，确保稳定性与可控性
- 外挂生态：所有业务逻辑以独立插件形式开发，严格遵循「六大挂件分类体系」
- 黑盒交付：关键模块编译为 DLL/EXE 黑盒，保护核心逻辑不泄露
- 全量卷宗归档：所有操作、修改与讨论均永久记录，实现100%可追溯

## 主要功能规划
1.  **族谱资料管理**：支持多版本族谱图片上传、人工校对与差异记录
2.  **角色化权限控制**：超级管理员、财务专员、网格员三级隔离界面
3.  **数据持久化与导出**：自动保存数据，支持一键导出 Excel
4.  **零信任安全体系**：ECDSA/HMAC 验签、内核态 FSM 锁、SQLite 加密
